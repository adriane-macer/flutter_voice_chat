# voice_chat

A sample implementation of Voice chat using flutter framwork

## Getting Started

# Add required packages and plugins

# packages and plugins
- shared_preferences: ^2.2.2
- http: ^1.2.1
- path_provider: ^2.1.2
- cupertino_icons: ^1.0.6
- flutter_webrtc: ^0.9.48+hotfix.1
- stream_video_flutter: ^0.3.4
- flutter_dotenv: ^5.1.0
- record: ^5.0.4

# [Create account in Stream](https://getstream.io/)

- Create app on stream 
- Create user token for testing
- create .env.secret file and put in the root directory of the project 

# add the following text
API_KEY: <api key from the stream>
USER_TOKEN: <user token from the stream>
